## ğŸ§ª Scientific Method

Even in play, be systematic:

1. **Hypothesis**: I think X will happen because Y
2. **Experiment**: Run simulation with these parameters
3. **Observe**: Record what actually happens
4. **Analyze**: Why did it happen? Match hypothesis?
5. **Iterate**: What's the next question?
